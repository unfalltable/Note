## 基础概念

### CPU性能指标

- CPU时钟周期：一个CPU内数字脉冲信号振荡的周期
- CPU主频：CPU内数字脉冲信号振荡的频率
- CPI：平均每条指令执行需要的CPU时钟周期
- IPS：每秒执行多少条指令
- FLOPS：每秒执行多少次浮点运算
- 数据通路带宽：数据总线一次所能并行传送信息的位数
- 吞吐量：单位时间内处理请求的数量
- 响应时间：计算机发送一个请求到系统做出响应的时间
- 单位：K = 千 = 10^3^、M = 百万 = 10^6^、G = 十亿 = 10^9^、T = 万亿 = 10^12^

### 计算

- CPU时钟周期 = 指令条数 * CPI
- CPU主频 = 1 / CPU时钟周期
- IPS = CPU主频 / CPI
- 每条指令耗时 = CPI * CPU时钟周期
- CPU执行时间 = CPU时钟周期 / CPU主频 = 指令条数 * CPI / CPU主频
